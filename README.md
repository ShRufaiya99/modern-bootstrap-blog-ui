# ✍️ Modern Blog Website (Bootstrap 5)

This is a fully responsive blog landing page built using the **Bootstrap 5** framework. It demonstrates how to create structured layouts and reusable UI components for content-heavy websites.

### ✨ Key Features:
- **Responsive Grid:** Using the Bootstrap Grid system for a seamless experience on all devices.
- **Navbar & Footer:** Clean, functional navigation and informational footer.
- **Article Layouts:** Professional card designs for blog posts with hover effects.
- **Sidebar Components:** Categories, search bars, and recent post widgets.
- **Modern Typography:** Utilizing Bootstrap's typography classes for better readability.

### 🛠️ Technologies Used:
- **HTML5:** For semantic structure.
- **Bootstrap 5:** For rapid, responsive UI development.
- **CSS3:** For custom styling and fine-tuning the design.
